# Run on Metacentrum

Get the scripts to run
```bash
mkdir -p ~/projects
cd ~/projects/
git clone https://github.com/mjirik/tutorials.git
```


Run the experiment
```bash
cd ~/projects/tutorials/metacentrum/mmdetection_jupyterlab/
qsub Jupyter_job_22.01-r2.sh
```
